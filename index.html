개인정보 처리방침

본 서비스(이하 "서비스")는 이용자의 개인정보를 소중하게 생각하며 관련 법령을 준수합니다. 아래와 같이 개인정보 처리방침을 공개하여 이용자가 언제든지 확인할 수 있도록 합니다.

제1조 (수집하는 개인정보의 항목)

서비스는 다음과 같은 개인정보 또는 기기 정보를 수집할 수 있습니다.

1. 회원가입 및 기본 기능 이용 시

  • 이메일 주소(선택 기능 사용 시)
  • 기기 정보(운영체제, 앱 버전 등)
  • 오류 로그 및 분석 데이터(선택 시 Firebase Crashlytics 등)

2. 위치 기반 서비스 이용 시

  • 위치 정보(좌표 등)
  • 지도 검색 및 주변 시설 정보 제공을 위한 요청 데이터

3. 갤러리에서 이미지 불러오기 기능 이용 시

  • 사용자가 직접 선택한 이미지 파일 1개

※ 서비스는 이용자가 선택하지 않은 사진·파일에는 접근하지 않으며, Android 13 이상에서는 Photo Picker를 사용하여 갤러리 전체 접근 권한을 요구하지 않습니다.

제2조 (개인정보의 수집 및 이용 목적)

서비스는 다음과 같은 목적을 위해 개인정보를 수집·이용합니다.

  • 기본 기능 제공 및 사용자 맞춤 서비스 운영
  • 오류 분석 및 안정적인 서비스 제공
  • 지도 기반 운동시설 검색 기능 제공
  • 프로필 이미지 등 앱 내 이미지 표시 기능 제공(갤러리 선택 시)

제3조 (개인정보의 제3자 제공)

서비스는 이용자의 개인정보를 법령에 근거하지 않는 이상 제3자에게 제공하지 않습니다.

제4조 (외부 서비스(제3자 서비스)의 이용)

서비스는 아래와 같이 외부 서비스를 이용하여 일부 기능을 제공합니다.

1. Google 계정 연동 (현재 비활성화)

서비스 내부에 Google 로그인 기능이 포함되어 있으나, 현재 사용자에게 제공되지 않으며 비활성화된 상태입니다. 따라서 Google LLC로 어떠한 개인정보도 전송되지 않습니다.

  • 제공받는 자: Google LLC
  • 제공되는 항목: 없음 (기능 비활성화)
  • 제공 목적: 없음
  • 보유·이용 기간: 없음

2. Google Maps API 활용

  • 제공받는 자: Google LLC
  • 제공되는 정보: 위치 정보(좌표 등), 검색 요청 데이터
  • 제공 목적: 지도 표시 및 주변 운동 시설 검색 기능 제공
  • 보유·이용 기간: Google Maps API 약관에 따름

제5조 (개인정보 처리의 위탁)

서비스는 일부 기능을 아래 업체에 위탁하여 처리할 수 있습니다.

  • 수탁 업체: Google Firebase (Google LLC)
  • 위탁 업무 내용:
    • 오류 로그 및 진단 데이터 수집·분석
    • 앱 안정성 개선을 위한 통계 처리
  • 보유 및 이용 기간: 계정 삭제 또는 동의 철회 시까지

서비스는 위탁 계약 체결 시 이용자의 개인정보가 안전하게 관리되도록 감독합니다.

제6조 (이용자의 권리와 행사 방법)

이용자는 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 정정·삭제 요청
  • 개인정보 처리 정지 요청
  • 동의 철회

위 권리 행사는 앱 내 문의 또는 아래 기재된 이메일을 통해 요청할 수 있습니다. 서비스는 법령에 따라 즉시 처리합니다.

제7조 (개인정보의 파기 절차 및 방법)

1. 파기 절차

  • 이용자가 앱 삭제 또는 계정 삭제를 요청하면 관련 정보를 복구 불가능한 방법으로 삭제합니다.
  • 법령상 보관 의무가 없는 경우 즉시 파기합니다.

2. 파기 방법

  • 전자 파일: 복구 불가능한 기술적 조치 적용
  • 종이 문서(해당 시): 분쇄 또는 소각

제8조 (만 14세 미만 아동의 개인정보 처리)

서비스는 만 14세 미만을 대상으로 설계된 기능을 제공하지 않으며, 고의적으로 개인정보를 수집하지 않습니다. 만약 수집 사실을 인지하면 지체 없이 삭제합니다.

제9조 (개인정보 보호를 위한 기술적·관리적 대책)

  • 개인정보 접근 권한 최소화
  • 보안 정책에 따른 데이터 보호 조치
  • 정기적인 업데이트 및 보안 패치
  • 개발 단계에서 개인정보 최소 수집 설계
